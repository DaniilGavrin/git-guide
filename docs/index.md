# Git Guide

Добро пожаловать в памятку по командам Git! Этот ресурс создан для того, чтобы помочь новичкам и опытным разработчикам быстро находить нужные команды и понимать основы работы с системой контроля версий.

## Что вы найдете здесь

- **Основы**: Базовые команды для начала работы с Git.
- **Ветки**: Как создавать, переключаться и управлять ветками.
- **Удаленная работа**: Работа с удаленными репозиториями (GitHub, GitLab и др.).
- **Продвинутое**: Сложные сценарии, отмена изменений, ребейз и многое другое.

## Быстрый старт

Если вы только начинаете, рекомендуем начать с раздела [Основы](basic-commands.md).
