# Основные команды Git

Здесь собраны базовые команды, которые понадобятся вам для ежедневной работы с Git.

## Настройка пользователя

Перед началом работы необходимо настроить имя и email, которые будут отображаться в истории коммитов:

```bash
git config --global user.name "Ваше Имя"
git config --global user.email "your.email@example.com"
```

## Инициализация репозитория

Создание нового репозитория в текущей папке:

```bash
git init
```

Клонирование существующего репозитория:

```bash
git clone <url-репозитория>
```

## Проверка статуса

Просмотр изменений в рабочей директории:

```bash
git status
```

## Добавление файлов

Добавление конкретного файла в область индексации (staging area):

```bash
git add <файл>
```

Добавление всех измененных файлов:

```bash
git add .
```

## Создание коммита

Сохранение изменений с комментарием:

```bash
git commit -m "Описание изменений"
```

## Просмотр истории

Просмотр истории коммитов:

```bash
git log
```

Краткий просмотр истории (одна строка на коммит):

```bash
git log --oneline
```

## Отмена изменений

Отмена изменений в файле до последнего коммита (осторожно, изменения будут потеряны):

```bash
git checkout -- <файл>
```

Или в новых версиях Git:

```bash
git restore <файл>
```
