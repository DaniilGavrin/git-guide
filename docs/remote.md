# Удаленная работа

Работа с удаленными репозиториями позволяет синхронизировать ваш код с сервером (GitHub, GitLab, Bitbucket и др.) и сотрудничать с другими разработчиками.

## Добавление удаленного репозитория

Добавить удаленный репозиторий с именем `origin`:

```bash
git remote add origin <url-репозитория>
```

Просмотр списка удаленных репозиториев:

```bash
git remote -v
```

## Получение изменений

Загрузить изменения из удаленного репозитория без слияния:

```bash
git fetch origin
```

Загрузить изменения и сразу слить их с текущей веткой:

```bash
git pull origin <имя-ветки>
```

## Отправка изменений

Отправить локальную ветку в удаленный репозиторий:

```bash
git push origin <имя-ветки>
```

Отправить все локальные ветки:

```bash
git push --all origin
```

Удалить ветку на удаленном сервере:

```bash
git push origin --delete <имя-ветки>
```

## Клонирование

Скачать копию удаленного репозитория:

```bash
git clone <url-репозитория>
```

Клонировать в конкретную папку:

```bash
git clone <url-репозитория> <путь-к-папке>
```
